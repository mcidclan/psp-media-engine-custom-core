## PSP Media Engine Audio: Shared Double Buffer

Code sample demonstrating audio double buffer sharing between the Media Engine and the CPU, where the ME writes to the back buffer while the CPU reads from the front buffer.

## Usage
- Build it with CMake, then copy the EBOOT.PBP into a folder within the GAME directory.  
- Press Triangle to change audio generation mode.
